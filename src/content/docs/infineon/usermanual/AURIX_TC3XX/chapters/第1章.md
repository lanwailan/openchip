# 第一章：产品概述

## 1.1 产品简介

AURIX™ TC3xx 系列是英飞凌新一代的高性能微控制器，专为汽车和工业应用设计。该系列微控制器集成了多个 TriCore™ 内核，提供卓越的实时处理能力和安全性。

## 1.2 主要特点

- 多核 TriCore™ 架构
- 高达 300MHz 的内核频率
- 最大 6MB 闪存
- 348KB SRAM
- ASIL-D 硬件安全模块
- 多种通信接口

## 1.3 应用领域

- 动力总成控制
- 底盘控制系统
- 安全气囊系统
- 电动助力转向
- 电池管理系统
