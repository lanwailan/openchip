# 第二章：架构介绍

## 2.1 TriCore 架构

TriCore™ 是英飞凌专有的 32 位单片机架构，结合了 RISC 和 CISC 的优势，特别适合实时控制应用。

### 2.1.1 CPU 内核

- 32 位 RISC 处理器核心
- 支持超标量指令执行
- 内置浮点运算单元 (FPU)
- 紧密耦合存储器 (TCM)

### 2.1.2 存储器架构

- 程序闪存 (Flash)：最高 6MB
- 数据闪存 (DFLASH)：最高 128KB
- SRAM：最高 348KB
- 紧密耦合存储器 (TCM)：用于低延迟访问

## 2.2 安全特性

AURIX TC3xx 系列内置多种硬件安全机制：

- 内存保护单元 (MPU)
- 错误纠正码 (ECC)
- 硬件看门狗定时器
- 时钟监控电路
