# 第三章：外设功能

## 3.1 通用输入输出 (GPIO)

AURIX TC3xx 系列提供了丰富的 GPIO 引脚，支持多种配置模式：

- 输入/输出模式切换
- 内部上拉/下拉电阻
- 外部中断触发
- 复用功能选择

## 3.2 定时器模块

### 3.2.1 通用定时器 (GTM)

- 多通道 PWM 生成
- 捕获/比较单元
- 时间戳功能
- 事件计数

### 3.2.2 系统定时器 (STM)

- 高精度系统时间基准
- 多个定时器通道
- 中断生成功能

## 3.3 通信接口

- CAN-FD 接口：最多 6 个模块
- LIN 接口：支持主/从模式
- SPI 接口：高速串行通信
- ASC/UART 接口：异步串行通信
- Ethernet MAC：支持实时通信
