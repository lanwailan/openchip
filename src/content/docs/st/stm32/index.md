# STM32H7 系列用户手册

欢迎查阅 STMicroelectronics STM32H7 系列高性能微控制器的用户手册。

## 产品概述

STM32H7 系列是基于 ARM Cortex-M7 内核的高性能微控制器，具有以下特点：

- 最高 480MHz 的主频
- 双精度 FPU
- L1 高速缓存
- 丰富的模拟和数字外设
- 多种封装选项
