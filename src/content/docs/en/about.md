# About OpenChip

OpenChip is an open-source documentation hub dedicated to providing comprehensive technical documentation for semiconductor chips and microcontrollers.

## Our Mission

We aim to make chip documentation more accessible and organized for developers, engineers, and hobbyists worldwide. By providing a centralized platform for chip manuals, reference guides, and technical specifications, we help you find the information you need quickly and efficiently.

## Supported Manufacturers

We currently provide documentation for leading semiconductor manufacturers:

- **Infineon** - AURIX microcontrollers, power management chips, and automotive electronics
- **STMicroelectronics (ST)** - Microcontrollers, sensors, and power devices
- **NXP** - Automotive, industrial, and IoT solutions

## Features

- **Multi-language Support** - Documentation available in English and Chinese
- **Search Functionality** - Full-text search across all documentation
- **Organized Structure** - Clean, hierarchical organization of manuals and guides
- **Open Source** - Community-driven and freely accessible

## Contributing

We welcome contributions from the community. Whether you want to add new documentation, improve existing content, or help with translations, your contributions are valuable.
